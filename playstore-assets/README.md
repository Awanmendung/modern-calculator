#### How to Create Screenshots:

**Option 1: Take from Real Device**
1. Install the PWA on your Android phone
2. Open the calculator app
3. Take screenshots using phone's screenshot function
4. Transfer screenshots to computer
5. Save in `screenshots/` folder

**Option 2: Use Browser DevTools**
1. Open https://awanmendung.pythonanywhere.com in Chrome
2. Press F12 to open DevTools
3. Click the device toolbar icon (or Ctrl+Shift+M)
4. Select a phone device (e.g., Pixel 5)
5. Set dimensions to 1080 x 1920
6. Take screenshot: Ctrl+Shift+P ‚Üí "Capture screenshot"
7. Save in `screenshots/` folder

**Screenshot Ideas:**
- Screenshot 1: Calculator showing a simple calculation (e.g., 7+3=10)
- Screenshot 2: Calculator with history panel open
- Screenshot 3: Calculator showing complex calculation
- Screenshot 4: Calculator in different theme/state
